# سیستەمی حەلاقی — وەشانی ئۆفلاین (PWA بۆ ئایفۆن)

## ئەمە چییە؟
ئەم زیپە وەشانێکی تەواوی ئۆفلاینی سیستەمەکەتە. هیچ PHP، هیچ MySQL، هیچ سێرڤەر و هیچ ئینتەرنێتێکی پێویست نییە.
هەموو لۆجیک بە **Vanilla JavaScript** نووسراوەتەوە و هەموو داتا (خزمەتگوزاری، فرۆشتن، مەسرەف، ئەدمین) لەناو **IndexedDB**ی مۆبایلەکەت خۆی خەزن دەکرێت — نەک localStorage. ئەمە بەو واتایەیە کە سنووری ٥ مێگابایتی localStorage نییە، و دەتوانیت بە دڵنیایی سەدان هەزار تۆمار (فرۆشتن/مەسرەف) و چەندین گێگابایت داتا هەڵبگریت بەبێ کێشە — تەنها سنوورەکە بریتییە لە بۆشایی بەردەستی مۆبایلەکەت.

سیستەمەکە لە کرانەوەی یەکەمدا داوا لە Safari دەکات **Persistent Storage** بدات (`navigator.storage.persist()`)، بۆ ئەوەی کاتێک بۆشایی مۆبایلەکە کەم بووەوە، Safari خۆی داتاکانی ئەم ئەپە نەسڕیتەوە.

فایلەکان:
- `index.html` → هەموو ئەپەکە (login + dashboard + هەموو لۆجیک) لە یەک فایلدا
- `manifest.json` → بۆ ئەوەی وەک ئەپ لەسەر ئایفۆن دابمەزرێت
- `sw.js` → Service Worker بۆ کارکردنی تەواو ئۆفلاین (کاش کردن)
- `icons/` → ئایکۆنەکانی ئەپ

## هەنگاوەکانی دامەزراندن لەسەر ئایفۆن

1. هەموو فۆڵدەری `pwa` (یان ناوەرۆکی ئەم زیپە) هەڵبگرە لە شوێنێک کە بتوانیت لە ئینتەرنێت بەردەستی بکەیت — باشترین ڕێگا:
   - بارگەی فایلەکان بکە سەر **iCloud Drive** یان هەر ه�ۆستینگێکی سادە (وەک GitHub Pages، یان تەنانەت Dropbox).
   - یان لە کۆمپیوتەرەکەت ساده‌ترین ڕێگا: لوکاڵ سێرڤەرێکی بچووک ڕاکە (`python3 -m http.server`) و لە هەمان وایفای فایلەکە بکەرەوە لە Safari لەسەر ئایفۆن — تەنها بۆ یەک جار، دوای ئەوە کاتێک Cache بوو پێویستی بە ئینتەرنێت نامێنێت.
   - یان ئاسانترین ڕێگا: فایلی `index.html`، `manifest.json`، `sw.js` و فۆڵدەری `icons` بنێرە بۆ خۆت لە iMessage/Mail و لە ئایفۆن کلیکی لەسەر بکە بۆ کردنەوەی ڕاستەوخۆ لە Safari (لازمە هەموو فایلەکان لە یەک فۆڵدەر بن، بۆیە باشترین هەڵبژاردن iCloud Drive یەکێک لە فۆڵدەرە).

2. لە **Safari**ی ئایفۆن، فایلی `index.html` بکەرەوە.

3. دوگمەی **Share** (دڵنیاکردنەوە) بدە → **Add to Home Screen** هەڵبژێرە.

4. ناوی ئەپ دەکەوێتە سەر مۆبایلەکەت وەک ئەپێکی ڕاستەقینە (بێ ناونیشانی Safari)، بە ئایکۆنی خۆی.

5. لەمەودوا تەنها لەسەر ئایکۆنەکە کلیک بکە — هیچ ئینتەرنێتێکی پێویست نییە.

> **تێبینی گرنگ:** Safari لازمە یەکەم جار فایلەکە بکرێتەوە لە ڕێگەی http(s) (نەک کرداری فایلی لوکاڵی `file://`)، چونکە Service Worker و iCloud-based "Add to Home Screen" باشتر کاردەکات کاتێک لە http(s) دێت. ئەگەر هیچ سێرڤەرێکت نییە تەنها بۆ یەک جار، دەتوانیت یەکێک لەم بژاردانە بەکاربهێنیت:
> - لە کۆمپیوتەر: `python3 -m http.server 8000` لەناو فۆڵدەری pwa، پاشان لە ئایفۆن (هەمان وایفای) بڕۆ بۆ `http://[IP-ی-کۆمپیوتەرەکەت]:8000`
> - یان فایلەکان باربکە سەر هەر هۆستینگێکی فایلی خۆڕایی (GitHub Pages, Netlify Drop, و هتد) — تەنها بۆ یەک جار پێویستی پێیەتی، دوای ئەوە تەواو ئۆفلاین کاردەکات.

## چوونەژوورەوەی سەرەتایی
- ناوی بەکارهێنەر: `admin`
- پاسۆرد: `password`

دواتر لە بەشی **ڕێکخستنەکان** دەتوانیت ناو و پاسۆردی بگۆڕیت.

## داتاکانت چۆن خەزن دەکرێن؟
هەموو خزمەتگوزاری، فرۆشتن و مەسرەفەکان لەناو **IndexedDB**ی Safari/PWA لەسەر هەمان مۆبایل خەزن دەکرێن — کاتێک ئەپ دادەخرێت یا مۆبایل دەکوژێتەوە داتاکان نامێنن. ئەم شێوازە دەتوانێت بەهۆیەوە چەندین گێگابایت داتا (هەزاران تۆماری فرۆشتن و مەسرەف) هەڵبگریت بەبێ کێشە، بەپێچەوانەی localStorage کە سنووری نزیکەی ٥ مێگابایتی هەبوو.

تەنها ئەگەر History/Website Data ی Safari بسڕیتەوە یان ئەپەکە لابدەیت، داتاکان دەسڕێنەوە — بۆیە سیستەمەکە خۆی داوای **Persistent Storage** دەکات لە Safari بۆ کەمکردنەوەی ئەو مەترسییە.

⚠️ پێشنیار: کات بەکات (بۆ نموونە مانگانە) لە بەشی Settings → Export (ئەگەر داواکار بووی زیادی بکەین) یان بە دەستی screenshot لە ڕاپۆرتەکان وەربگرە، چونکە ئەمە تەنها لەناو مۆبایلەکەتدا خەزن دەکرێت و هیچ هەڵگرتنەوەی ئۆتۆماتیکی ناوەوەی نییە.

## گۆڕانکارییە سەرەکیەکان لە وەشانی کۆن
| پێشتر | ئێستا |
|---|---|
| PHP + PDO/MySQL | Vanilla JavaScript + IndexedDB |
| Session لە سێرڤەر | Session لە localStorage/sessionStorage (تەنها زانیاری چوونەژوورەوە، نەک داتای کارەکان) |
| bcrypt (`password_hash`) | SHA-256 (Web Crypto API) — تەنها بۆ بەکارهێنانی لۆکاڵی |
| فایلی جیاواز بۆ هەر بەش (index.php, login.php, api/*.php) | یەک فایلی `index.html` |
| پێویستی بە سێرڤەر و ئینتەرنێت | کاری تەواو ئۆفلاین، PWA بۆ ئایفۆن |
| سنووری خەزنکردن نزیکەی ٥ مێگابایت | بێ سنووری دیاریکراو (IndexedDB) — توانای هەڵگرتنی گێگابایتان داتا |
